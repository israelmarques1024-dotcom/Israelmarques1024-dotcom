## 👨‍💻 Sobre mim

Tenho **12 anos** e estudo desenvolvimento web, back-end, bancos de dados e aplicações desktop.

Gosto de construir projetos reais para aprender como cada parte de um sistema funciona. Meu objetivo não é apenas fazer o código rodar: quero entender a arquitetura, encontrar erros, melhorar a segurança e organizar projetos de forma profissional.

Atualmente, trabalho principalmente com **JavaScript**, **Node.js**, **Express** e **SQLite**, além de estudar **Electron**, Linux, Termux e ferramentas de terminal.

---

- Desenvolvimento de interfaces com **HTML, CSS e JavaScript**
- Criação de APIs e servidores com **Node.js e Express**

- Aplicações desktop com **Electron**
